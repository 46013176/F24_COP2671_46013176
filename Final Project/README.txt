Game Play: Run around the maze where three keys will spawn. Run into each key to collect it. After three keys are collected run back to the start.
 and touch the green zone to win. Avoid Traps, and escape before time runs out!

Controls: 

WASD - Basic Tank Movement
Space - Jump
ESC - Pause/Unpause
Enter - When prompted, restart game
